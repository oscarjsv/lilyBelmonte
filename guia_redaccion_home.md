# Guía de Referencia para Redacción - Página de Inicio (Home)

Este documento sirve como guía para crear los textos (copys) de la página de inicio. Por favor, trate de mantenerse dentro de los rangos de caracteres sugeridos para asegurar que el diseño del sitio web se mantenga perfecto en todos los dispositivos.

---

## 1. Hero (Sección Principal Arriba)

_Lo primero que ven los usuarios._

| Elemento             | Texto Actual (Ejemplo)                                                                  | **Rango Recomendado**  | Notas                                                    |
| :------------------- | :-------------------------------------------------------------------------------------- | :--------------------- | :------------------------------------------------------- |
| **Título Principal** | `La Ciencia de la Piel, Elevada a Arte`                                                 | **30 - 45 caracteres** | Debe ser impactante y corto.                             |
| **Subtítulo**        | `Descubre la armonía perfecta entre medicina estética avanzada y tu belleza auténtica.` | **70 - 90 caracteres** | Explica la propuesta de valor. Máximo 2 líneas en móvil. |
| **Botón (CTA)**      | `Descubre nuestros tratamientos`                                                        | **20 - 30 caracteres** | Verbo de acción claro.                                   |

## 2. Sección Servicios (Tarjetas/Cards)

_El grid donde se muestran los tratamientos._

### Cabecera de la Sección

| Elemento           | Texto Actual (Ejemplo)                          | **Rango Recomendado**  | Notas            |
| :----------------- | :---------------------------------------------- | :--------------------- | :--------------- |
| **Título Sección** | `Nuestros Tratamientos`                         | **15 - 25 caracteres** | Corto y directo. |
| **Subtítulo**      | `Protocolos especializados para cada necesidad` | **40 - 55 caracteres** | Breve contexto.  |

### Contenido de las Tarjetas (Para cada servicio)

_Es crucial mantener estos textos similares en longitud para que las tarjetas se vean alineadas._

| Elemento            | Texto Actual (Ejemplo)                                                   | **Rango Recomendado**  | Notas                                                                              |
| :------------------ | :----------------------------------------------------------------------- | :--------------------- | :--------------------------------------------------------------------------------- |
| **Título Servicio** | `Luminosidad y Juventud Eterna`                                          | **20 - 35 caracteres** | Nombre atractivo del servicio.                                                     |
| **Descripción**     | `Protocolos de vanguardia para una piel radiante que desafía el tiempo.` | **65 - 85 caracteres** | **Muy Importante:** Si pasas de 90 caracteres, la tarjeta se desalinea en móviles. |
| **Texto Enlace**    | `Ver detalles`                                                           | **10 - 15 caracteres** | Corto.                                                                             |
